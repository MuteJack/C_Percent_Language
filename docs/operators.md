# C% 연산자

## 산술 연산자

| 연산자 | 설명   | 피연산자 타입          | 결과 타입          | 예시          |
| ------ | ------ | ---------------------- | ------------------ | ------------- |
| `+`    | 덧셈   | 정수, 실수             | 피연산자와 동일    | `3 + 4` → 7  |
| `-`    | 뺄셈   | 정수, 실수             | 피연산자와 동일    | `7 - 2` → 5  |
| `*`    | 곱셈   | 정수, 실수             | 피연산자와 동일    | `3 * 4` → 12 |
| `/`    | 나눗셈 | 정수, 실수             | 피연산자와 동일    | `7 / 2` → 3  |
| `%`    | 나머지 | 정수, 실수             | 피연산자와 동일    | `7 % 3` → 1  |
| `**`   | 제곱   | 정수, 실수             | 피연산자와 동일    | `2 ** 10` → 1024 |
| `/%`   | 몫+나머지 | 정수, 실수          | 배열 `[몫, 나머지]` | `17 /% 5` → `[3, 2]` |
| `-`    | 부호   | 정수, 실수 (단항)      | 피연산자와 동일    | `-x`          |

- 정수 `/`는 정수 나눗셈 (소수점 버림)
- 실수 `%`는 `fmod` 사용
- `**`는 우결합: `2 ** 3 ** 2` = `2 ** 9` = `512`
- `**` 음수 지수: `2 ** -1` = `0.5` (float 변환)
- `int` 타입 산술 시 int64 오버플로 → BigInt 자동 승격
- `+`는 문자열 연결에도 사용 가능 (`"hello" + " world"`)
- `/%`는 나눗셈의 몫과 나머지를 동시에 반환 (`divmod`와 동일)

### 0으로 나누기 (Division by Zero)

`/`, `%`, `/%` 연산에서 오른쪽 피연산자가 0이면 런타임 에러 발생:

```
int x = 10 / 0;     // 에러: Division by zero
int y = 10 % 0;     // 에러: Modulo by zero
int z = 10 /% 0;    // 에러: Division by zero in /%
```

- 정수, 실수 모두 동일하게 에러
- BigInt 나눗셈도 동일

## 증감 연산자

| 연산자 | 설명       | 피연산자 타입     | 예시   |
| ------ | ---------- | ----------------- | ------ |
| `++`   | 1 증가     | 정수, 실수, char  | `x++`  |
| `--`   | 1 감소     | 정수, 실수, char  | `x--`  |

- 전위(`++x`) / 후위(`x++`) 모두 지원
- 전위: 증감 후 값 반환
- 후위: 증감 전 값 반환
- sized 정수/char: 오버플로 시 wrap-around

## 대입 연산자

| 연산자 | 설명        | 예시       | 동등한 표현     |
| ------ | ----------- | ---------- | --------------- |
| `=`    | 대입        | `x = 5`    |                 |
| `+=`   | 더하고 대입 | `x += 3`   | `x = x + 3`    |
| `-=`   | 빼고 대입   | `x -= 3`   | `x = x - 3`    |
| `*=`   | 곱하고 대입 | `x *= 3`   | `x = x * 3`    |
| `/=`   | 나누고 대입 | `x /= 3`   | `x = x / 3`    |
| `%=`   | 나머지 대입 | `x %= 3`   | `x = x % 3`    |

## 비교 연산자

| 연산자 | 설명        | 결과 타입 | 예시           |
| ------ | ----------- | --------- | -------------- |
| `==`   | 같음        | `bool`    | `x == y`       |
| `!=`   | 다름        | `bool`    | `x != y`       |
| `<`    | 작음        | `bool`    | `x < y`        |
| `>`    | 큼          | `bool`    | `x > y`        |
| `<=`   | 작거나 같음 | `bool`    | `x <= y`       |
| `>=`   | 크거나 같음 | `bool`    | `x >= y`       |

- 정수끼리 비교 시 BigInt 비교 지원
- 문자열끼리 `==`, `!=` 비교 가능
- 타입이 다르면 숫자로 변환 후 비교

### 연쇄 비교 (Chained Comparison)

`<`, `>`, `<=`, `>=` 비교 연산자를 체이닝할 수 있다:

```
0 < x < 10         // (0 < x) && (x < 10)
0 <= x <= 100      // (0 <= x) && (x <= 100)
10 > x > 0         // (10 > x) && (x > 0)
0 < a < b < c < 100  // 3개 이상 체이닝 가능
```

- `<`, `<=`, `>`, `>=`만 체이닝 가능 (`==`, `!=`는 불가)
- 중간 표현식은 한 번만 평가 (부작용 안전)
- `for`문과 결합 가능: `for(0 <= i < 10) { ... }`

## 논리 연산자

| 연산자 | 설명   | 피연산자 타입 | 결과 타입 | 예시          |
| ------ | ------ | ------------- | --------- | ------------- |
| `&&`   | AND    | 모든 타입     | `bool`    | `a && b`      |
| `\|\|` | OR     | 모든 타입     | `bool`    | `a \|\| b`    |
| `!`    | NOT    | 모든 타입     | `bool`    | `!a`          |

- 모든 값은 bool로 변환 가능 (truthy/falsy)
- `0`, `0.0`, `false`, `""`, `[]` → `false`
- 그 외 → `true`

## 문자열 연산자

| 연산자 | 설명       | 예시                         |
| ------ | ---------- | ---------------------------- |
| `+`    | 문자열 연결 | `"hello" + " " + "world"`  |

- 문자열 리터럴(`"..."`) 또는 `string` 변수에 `+`를 사용하면 항상 문자열 연결
- 한쪽이 문자열이면 다른 쪽을 자동 변환 후 연결
- `"hello" + 2` → `"hello2"` (자동 변환)
- `"x = " + string(42)` → `"x = 42"`
- `char` 변수의 `+`는 산술 연산 (문자열 연결 아님)

## 삼항 연산자 (Ternary Operator)

```
조건 ? 참값 : 거짓값
```

```
int x = 10;
int y = 20;
int max = x > y ? x : y;       // 20

// 중첩 (우결합)
int score = 85;
string grade = score >= 90 ? "A" : score >= 80 ? "B" : score >= 70 ? "C" : "F";
// → "B"

// 표현식 내에서 사용
int val = (x > 5 ? 10 : 0) + (y > 10 ? 20 : 0);  // 30
```

- C/C++과 동일한 `조건 ? 참 : 거짓` 문법
- 우결합: `a ? b : c ? d : e` = `a ? b : (c ? d : e)`
- 조건이 true면 참값만, false면 거짓값만 평가 (short-circuit)

## 연산자 우선순위

| 우선순위 | 연산자                     | 결합 방향 |
| -------- | -------------------------- | --------- |
| 1 (최고) | `()` `[]` `++`후위 `--`후위 | 좌→우    |
| 2        | `-`단항 `!` `++`전위 `--`전위 | 우→좌   |
| 3        | `**`                       | 우→좌    |
| 4        | `*` `/` `%` `/%`           | 좌→우    |
| 5        | `+` `-`                    | 좌→우    |
| 6        | `<` `>` `<=` `>=`          | 좌→우    |
| 7        | `==` `!=`                  | 좌→우    |
| 8        | `&&`                       | 좌→우    |
| 9        | `\|\|`                     | 좌→우    |
| 10       | `? :`                      | 우→좌    |
| 11 (최저) | `=` `+=` `-=` `*=` `/=` `%=` | 우→좌   |
