# C% 파일 I/O 내장 함수

> 구현 예정

## 개요

파일 읽기/쓰기를 위한 내장 함수. C++의 `<fstream>`과 Python의 `open()`을 참고하여 설계 예정.

## 예정 기능

### 파일 열기/닫기

| 함수 | 설명 | 비고 |
| --- | --- | --- |
| `fopen(path, mode)` | 파일 열기 | mode: `"r"`, `"w"`, `"a"` 등 |
| `fclose(f)` | 파일 닫기 | |

### 읽기

| 함수 | 설명 | 비고 |
| --- | --- | --- |
| `fread(f)` | 파일 전체 읽기 (string 반환) | |
| `freadline(f)` | 한 줄 읽기 | |
| `freadlines(f)` | 전체를 줄 단위 배열로 반환 | |

### 쓰기

| 함수 | 설명 | 비고 |
| --- | --- | --- |
| `fwrite(f, data)` | 파일에 쓰기 | |
| `fwriteline(f, data)` | 줄바꿈 포함 쓰기 | |

### 유틸리티

| 함수 | 설명 | 비고 |
| --- | --- | --- |
| `fexists(path)` | 파일 존재 여부 | bool 반환 |

## 설계 방향

- C 스타일 함수형 인터페이스 (`fopen`, `fclose`, `fread`, `fwrite`)
- 모드 문자열: `"r"` (읽기), `"w"` (쓰기, 덮어쓰기), `"a"` (추가)
- 파일 핸들은 내부적으로 정수 ID 또는 별도 타입으로 관리
