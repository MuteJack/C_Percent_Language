# C% 수학 내장 함수

## `divmod(a, b)`

나눗셈의 몫과 나머지를 배열로 반환한다.

```
int result[] = divmod(17, 5);
print(result);           // [3, 2]

int result2[] = divmod(100, 7);
print(result2);          // [14, 2]
```

- 정수, BigInt, 실수 모두 지원
- `b == 0`이면 에러 (0으로 나누기)
- 반환 타입: `[몫, 나머지]` (2원소 배열)
- `/%` 연산자와 동일: `17 /% 5` == `divmod(17, 5)` == `[3, 2]`
