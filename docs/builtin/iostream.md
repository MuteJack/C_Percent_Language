# C% I/O 내장 함수

## `print(expr, ...)`

값을 표준 출력에 출력한다. **줄바꿈 없음.** (구문으로 구현됨)

```
print(42);               // 42 (줄바꿈 없음)
print("hello");          // hello
print(1, 2, 3);          // 1 2 3
print(arr);              // [1, 2, 3]
```

## `println(expr, ...)`

값을 표준 출력에 출력한다. **줄바꿈 포함.** (구문으로 구현됨)

```
println(42);             // 42\n
println("hello");        // hello\n
println(1, 2, 3);        // 1 2 3\n
println();               // \n (빈 줄)
```

- 여러 인자는 공백으로 구분
- `char` 타입은 문자로 출력
- 배열은 `[원소1, 원소2, ...]` 형태로 출력
- 문자열 배열은 `["a", "b"]` 형태로 출력
- dict는 `{"key": value, ...}` 형태로 출력

## `input()`

표준 입력에서 한 줄을 읽어 문자열로 반환한다. (구문으로 구현됨)

```
string name = input();
int age = int(input());
```

- 항상 문자열로 반환 (숫자가 필요하면 `int()`, `float()` 등 타입 캐스트 사용)
- 인자 없음
